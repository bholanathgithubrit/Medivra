# 🏥 Medivra - AI Medical Assistant Bot

Medivra is a smart AI-powered medical assistant designed to help users with healthcare-related queries, basic diagnosis support, and medical guidance. It aims to simplify access to healthcare information and improve user experience through intelligent automation.

---

## 🚀 Features

* 🤖 AI-powered medical query handling
* 💊 Medicine suggestions & basic guidance
* 📅 Appointment assistance (future scope)
* 📊 Health tips and recommendations
* 🔍 Symptom-based assistance
* 🔐 Secure and scalable backend

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB

### AI / ML (Planned / Optional)

* OpenAI / LLM integration
* NLP-based symptom analysis

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/bholanathgithubrit/Medivra.git
cd Medivra
```

---

### 2. Create virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
```

---

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Run the application

```bash
python app.py
```

---

## 🔐 Environment Variables

Create a `.env` file and add:

```
API_KEY=your_api_key
DB_URI=your_database_url
```
